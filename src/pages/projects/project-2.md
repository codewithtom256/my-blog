## Habit Tracker

A simple web application to help users track and visualize their daily habits. Built with Astro and TypeScript.

### Features
- Add, edit, and delete habits
- Daily progress charts
- Responsive design

### Technologies
- Astro
- TypeScript
- Chart.js
